## üìè **CALCULATION LOGIC FOR QUANTITIES**

### **DC Structure Earthing**
- **DC Bonding Jumpers**: 1 per table = `totalNumberOfTables`
- **DC PE Cable**: Adiabatic sizing (Ik=Isc, t=1s, k=115), length = 5m per table
- **DC Earth Pits**: 1 per 20 tables = `ceil(totalNumberOfTables √∑ 20)`

### **Lightning Protection**
- **ESE Lightning Arrestors**: `ceil(totalPlantAreaM2 √∑ 10,000)`
- **LA Earth Pits**: 3 per ESE LA = `3 √ó number of LAs`
- **Earthing Compound**: 25kg per pit = `25 √ó total pits`

### **AC Earthing PE Cable Sizing**
- **LV PE**: IEC Table 54.2 rules (‚â§16mm¬≤ ‚Üí same, 16-35mm¬≤ ‚Üí 16mm¬≤, >35mm¬≤ ‚Üí phase√∑2)
- **HV PE**: Adiabatic (Ik=10kA, t=1s, k=115)
- **Length**: Same as phase cable route length

### **Equipment Bonding**
- **LV**: 6mm¬≤, 2m each, 1 per inverter + 1 per combiner
- **HV**: 16mm¬≤, 3m each, 2 per inverter skid/IDT/PT

### **Earth Grid Calculation**
- **Grid Size**: LV 30√ó30m (900m¬≤), HV 40√ó40m (1600m¬≤)
- **Strip Length**: Perimeter + cross strips at 10m spacing
- **Rods**: Every 20m + 4 corner rods, additional if Rg > target

### **Current Transformers**
- **LV CT Primary**: 1.2 √ó breaker rating
- **HV CT Primary**: 1.25 √ó rated current
- **Quantity**: 3 per feeder (one per phase)

### **Potential Transformers**
- **HV PTs**: 3 per voltage level (one per phase)
- **Quantity**: Varies by system complexity

### **Surge Protective Devices**
- **LV SPDs**: Type 2, quantity based on panels
- **HV SPDs**: ZnO arresters, 3 per voltage level

### **Protection Relays**
- **LV**: 1 per panel
- **HV**: 1 per feeder, additional for transformer protection

### **Communication Cables**
- **Length**: Route length + 10% slack
- **RS-485**: Inverter to SCADA
- **Fiber**: Yard to substation

### **Busbars**
- **LV**: 1.5 √ó incomer current rating
- **HV**: 1.25 √ó rated current, 31.5kA withstand

### **Transformer Earthing**
- **4 pits per transformer**: 2 neutral + 2 body
- **25kg compound per pit**
